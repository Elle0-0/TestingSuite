## API Client: First Iteration - Data Collection Client

A development team is building a monitoring tool that gathers sensor readings from a number of weather stations distributed across a country.

Each station exposes an online endpoint that returns its latest reading. The program is given a collection of station endpoints and must contact each one, retrieve the data and combine the results into a single in-memory collection.

For this initial version, you may assume that all stations respond correctly and return valid data.

Your program must return:

- The collection of successfully retrieved station readings
- Any relevant summary information needed to verify correctness

The emphasis at this stage is correctness and clear structure.
